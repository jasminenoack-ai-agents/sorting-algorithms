# sorting-algorithms

Visual demonstrations of common sorting algorithms.

## Requirements

This project requires Node.js version 16 or higher. The requirement is
enforced via the `engines` field in `package.json`.

## Installation

Install the dependencies:

```bash
npm install
```

## Building

Compile the project and watch for changes with:

```bash
npm run build
```

This command runs `webpack --watch` internally.

## Running tests

Execute the Karma test suite:

```bash
npm test
```
